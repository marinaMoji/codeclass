# Google's Python Class - Jupyter Notebooks

This directory contains Jupyter notebook versions of the course materials from [Google's Python Class](https://developers.google.com/edu/python/). The notebooks are organized into two types:

1. **Lesson Notebooks** - Tutorial content with explanations and examples (00-06)
2. **Exercise Notebooks** - Practice exercises to reinforce learning (01-03 with "Exercises" suffix)

## Lesson Notebooks (Tutorial Content)

### 00_Python_Intro.ipynb
Introduction to Python basics:
- Variables and basic types
- Control flow (if/else, loops)
- Functions
- Basic syntax

### 01_Strings_Lesson.ipynb
String manipulation tutorial:
- String indexing and slicing
- String methods
- String formatting
- String operations

### 02_Lists_Lesson.ipynb
Lists tutorial:
- List creation and indexing
- List methods
- List operations
- Iteration

### 03_Sorting_Lesson.ipynb
Sorting tutorial:
- Basic sorting
- Custom sorting with key functions
- Multiple criteria sorting
- Sorting strings and complex data

### 04_Dicts_and_Files_Lesson.ipynb
Dictionaries and file operations:
- Dictionary basics and operations
- Iterating over dictionaries
- Reading and writing files
- File modes

### 05_Regular_Expressions_Lesson.ipynb
Regular expressions:
- Pattern matching
- Common regex patterns
- Regex functions (search, findall, sub, split)
- Groups and capturing

### 06_Utilities_Lesson.ipynb
Python utilities:
- Command-line arguments (sys)
- File and directory operations (os)
- URL handling (urllib)
- Standard library modules (random, datetime, json)

## Exercise Notebooks

### 01_Strings_Exercises.ipynb
Basic string manipulation exercises:
- `donuts()` - Format strings with conditional logic
- `both_ends()` - String slicing
- `fix_start()` - String replacement
- `mix_up()` - String concatenation and slicing

### 01b_Strings_Additional_Exercises.ipynb
Additional string exercises:
- `verbing()` - Conditional string modification
- `not_bad()` - Substring finding and replacement
- `front_back()` - String splitting and concatenation

### 02_Lists_Exercises.ipynb
Basic list manipulation exercises:
- `match_ends()` - List iteration and counting
- `front_x()` - List sorting with custom logic
- `sort_last()` - Sorting with custom keys

### 02b_Lists_Additional_Exercises.ipynb
Additional list exercises:
- `remove_adjacent()` - List filtering
- `linear_merge()` - Merging sorted lists

### 03_Wordcount_Exercise.ipynb
A comprehensive exercise combining multiple concepts:
- `print_words()` - Word counting with dictionaries
- `print_top()` - Finding most common words
- File I/O and command-line arguments

## How to Use

1. **Install Jupyter Notebook** (if not already installed):
   ```bash
   pip install jupyter notebook
   ```

2. **Launch Jupyter Notebook**:
   ```bash
   jupyter notebook
   ```

3. **Recommended Learning Path**:
   
   **Step 1: Study the Lessons** (in order):
   - `00_Python_Intro.ipynb` - Start here for Python basics
   - `01_Strings_Lesson.ipynb` - Learn string manipulation
   - `02_Lists_Lesson.ipynb` - Learn about lists
   - `03_Sorting_Lesson.ipynb` - Learn sorting techniques
   - `04_Dicts_and_Files_Lesson.ipynb` - Learn dictionaries and file I/O
   - `05_Regular_Expressions_Lesson.ipynb` - Learn regex patterns
   - `06_Utilities_Lesson.ipynb` - Learn useful Python utilities
   
   **Step 2: Practice with Exercises**:
   - `01_Strings_Exercises.ipynb` - Practice string exercises
   - `01b_Strings_Additional_Exercises.ipynb` - More string practice
   - `02_Lists_Exercises.ipynb` - Practice list exercises
   - `02b_Lists_Additional_Exercises.ipynb` - More list practice
   - `03_Wordcount_Exercise.ipynb` - Comprehensive exercise combining all concepts

4. **Work through each notebook**:
   - **Lesson notebooks**: Read explanations and run code examples
   - **Exercise notebooks**: Fill in function implementations and run tests
